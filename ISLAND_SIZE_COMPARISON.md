# 灵动岛尺寸对比

## 尺寸变化

| 状态 | 原始尺寸 | 新尺寸 | 缩小比例 |
|------|---------|--------|---------|
| 收缩 | 200x45 | 140x35 | 30% |
| 标准 | 300x50 | 200x40 | 33% |
| 展开 | 450x55 | 280x45 | 38% |

## 配置变化

| 参数 | 原始值 | 新值 | 说明 |
|------|--------|------|------|
| 默认缩放 | 1.0 | 0.85 | 整体缩小到85% |
| Y偏移 | 10px | 8px | 稍微靠近顶部 |
| 圆角半径 | 25px | 18px | 适配更小尺寸 |
| 内边距 | 12px | 10px | 减小内边距 |
| 图标文本间距 | 5px | 4px | 更紧凑 |
| 元素间距 | 8px | 6px | 更紧凑 |
| 背景透明度 | 0xF0 | 0xE8 | 稍微更透明 |
| 文本透明度 | 0.9 | 0.92 | 稍微提高 |

## 视觉效果

### 优点
- ✅ 更小巧，不遮挡视野
- ✅ 更符合"灵动"的设计理念
- ✅ 性能更好（使用纯色背景）
- ✅ 更稳定（无纹理加载问题）
- ✅ 更简洁的外观

### 背景渲染
- 使用纯色圆角矩形
- 平滑的边缘（GL_POLYGON_SMOOTH）
- 半透明黑色背景
- 可选阴影效果

## 如何调整

如果需要进一步调整尺寸，修改以下文件：

1. **IslandState.java** - 修改三种状态的宽高
2. **IslandConfig.java** - 修改默认缩放、圆角、间距等
3. **IslandRenderer.java** - 修改内边距和元素间距

## 启用调试模式

在配置文件中设置 `debugMode: true` 可以看到详细的布局信息：
```json
{
  "debugMode": true
}
```

配置文件位置：`config/examplemod/dynamic_island.json`
