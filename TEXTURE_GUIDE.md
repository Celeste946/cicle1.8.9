# 灵动岛纹理使用指南

## 纹理文件

### 位置
```
src/main/resources/assets/examplemod/textures/gui/ldd.png
```

### 要求
- **格式**: PNG（支持透明度）
- **推荐尺寸**: 256x64 像素（2的幂次方）
- **最小尺寸**: 128x32 像素
- **最大尺寸**: 512x128 像素

## 纹理设计建议

### 尺寸比例
- 宽高比应该接近灵动岛的比例（约 5:1 到 7:1）
- 标准状态：200x40 像素
- 纹理会被拉伸以适应不同状态

### 视觉效果
1. **圆角**: 在纹理四角设计圆角效果
2. **渐变**: 可以使用渐变色增加深度感
3. **透明度**: 边缘使用半透明效果更自然
4. **阴影**: 可以在纹理中内置阴影效果

### 颜色建议
- 深色背景（黑色、深灰）
- 半透明（Alpha 通道 200-240）
- 避免过于鲜艳的颜色
- 确保文本在上面清晰可读

## 渲染方式

### 当前实现
- 使用简单的纹理拉伸
- 纹理会被拉伸以填充整个灵动岛区域
- 如果纹理加载失败，自动降级到纯色背景

### 自动降级
如果纹理加载失败，系统会：
1. 在控制台输出错误信息
2. 自动切换到纯色圆角矩形背景
3. 使用配置的背景颜色（默认：0xE8000000）

## 调试

### 检查纹理是否加载
查看控制台输出：
```
[TextureValidator] 验证纹理: examplemod:gui/ldd
[TextureValidator] ✓ 纹理加载成功
[IslandRenderer] ✓ 背景纹理加载成功
```

### 常见问题

#### 纹理显示为白色方块
- 检查文件路径是否正确（区分大小写）
- 确认文件存在于正确位置
- 检查文件格式是否为 PNG

#### 纹理变形
- 确保纹理尺寸是 2 的幂次方
- 调整纹理的宽高比
- 考虑使用九宫格拉伸（高级功能）

#### 纹理不显示
- 查看控制台错误信息
- 使用 TextureValidator 验证路径
- 确认资源文件已正确打包

## 切换到纯色背景

如果不想使用纹理，可以：

1. 删除或重命名 `ldd.png` 文件
2. 系统会自动使用纯色背景
3. 在配置中调整背景颜色：
   ```json
   {
     "backgroundColor": -402653184
   }
   ```

## 性能考虑

- 纹理渲染性能良好
- 纯色背景性能略好
- 对于大多数情况，性能差异可忽略
- 如果遇到性能问题，可以切换到纯色背景

## 自定义纹理

要使用自定义纹理：

1. 创建符合要求的 PNG 文件
2. 替换 `src/main/resources/assets/examplemod/textures/gui/ldd.png`
3. 重新编译 mod
4. 纹理会自动加载
